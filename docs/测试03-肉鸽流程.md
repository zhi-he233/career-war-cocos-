# 测试 03 — 肉鸽流程

> 前置：服务器在运行，搭建 01-04 已完成

### 1. 创建肉鸽房间

- 打开 Home 场景预览
- 点 **肉鸽爬塔** 按钮

### 2. Lobby → Battle

- 选角色 → Start Game → 进入 Battle 场景

### 3. 战斗结束后

服务器将 phase 切换为肉鸽阶段，客户端自动跳到 Roguelite 场景。

Roguelite 场景的灰盒 UI 支持：

| 阶段 | 显示内容 |
|---|---|
| reward | 奖励三选一按钮 |
| roguelite_event | 事件描述 + AB 选项按钮 |
| roguelite_shop | 商店物品列表 + Leave Shop 按钮 |
| roguelite_rest | 休息点行动按钮（最多 5 个） |
| roguelite_continue | 路线节点按钮 + Finish Run 按钮 |

### 4. 状态栏

状态栏会显示当前 roguelite 阶段和 server ack 结果。
