# 搭建 06 - 首页桌面 UI 摆位

这一篇只讲 `Home.scene`。

现在首页按“桌面 + 卡片 + 骰子 + 规则书”的正式界面整理过了。运行时默认不会再自动生成房间号输入框、刷新房间、状态文字那些开发调试 UI。

## 当前节点

```text
Canvas
  Camera
  TableBackground
  ClassicButton
  PveButton
  ProfileButton
  RogueliteButton
  RuleBookButton
```

## 每个节点做什么

| 节点 | 图片 | 点击后 |
|---|---|---|
| TableBackground | 空白桌面 | 纯背景 |
| ClassicButton | 经典对战卡片 | 创建 classic 房间 |
| PveButton | 人机训练卡片 | 创建 pve_1v1 房间 |
| ProfileButton | 玩家档案卡片 | 打开玩家档案弹窗 |
| RogueliteButton | 骰子 | 播放骰子小动画，然后创建 pve_roguelite 房间 |
| RuleBookButton | 规则书 | 打开规则书弹窗 |

## 你怎么微调

1. 打开 `assets/scenes/Home.scene`
2. 在 Hierarchy 里点一个节点，比如 `ClassicButton`
3. 右边 Inspector 找 `Position`
4. 改 `X / Y`

Cocos 坐标很好记：

```text
X 变大：往右
X 变小：往左
Y 变大：往上
Y 变小：往下
```

要改大小，找同一个节点上的：

```text
UITransform -> Content Size -> Width / Height
```

要改旋转，改：

```text
Rotation
```

## 按钮事件怎么看

每张可点击图片都有 `Button` 组件。

在 Inspector 里展开：

```text
Button -> Click Events
```

你会看到目标是 `Canvas`，方法来自 `HomeScene`：

| 节点 | 方法 |
|---|---|
| ClassicButton | `createClassicRoom` |
| PveButton | `createPveRoom` |
| RogueliteButton | `createRogueliteRoom` |
| ProfileButton | `openProfile` |
| RuleBookButton | `openRuleGuide` |

如果以后你自己新建图片按钮，流程就是：

1. 新建 Node
2. 加 `UITransform`
3. 加 `Sprite`
4. 加 `Button`
5. `Button -> Click Events` 数量设成 1
6. 把 `Canvas` 拖进去
7. 选择 `HomeScene` 里的方法

## 调试 UI

`HomeScene` 里有一个字段：

```text
Create Debug Ui
```

平时保持关闭。

如果你以后想临时显示房间号输入框、刷新房间、状态文字，可以选中 `Canvas`，在 `HomeScene` 组件里勾上它。
