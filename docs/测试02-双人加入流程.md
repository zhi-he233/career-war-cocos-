# 测试 02 — 两个客户端加入同一房间

> 前置：服务器在运行，搭建 01-04 已完成

### 1. 窗口 A 创建房间

- 双击 `Home.scene`，点 ▶️ 预览 → 得到窗口 A
- 点 **经典混战**
- 进入 Lobby 后，记下状态栏显示的 Room ID（如 `Room abc123 | lobby`）

### 2. 窗口 B 加入房间

- 再开一个预览窗口（Cocos 菜单 → **预览** → **浏览器预览**，或直接再点 ▶️ 会新开标签页）

**方法一：输入 Room ID**

- 在 Home 场景的 Room ID 输入框里输入刚才记下的房间号
- 点 **Join Room** 按钮

**方法二：刷新房间列表**

- 点 **Refresh Rooms** 按钮
- 房间列表里应出现窗口 A 创建的房间
- 点那一行加入

### 3. 验证

- 窗口 B 应跳到同一个 Lobby 场景
- 两个窗口的玩家列表里应显示双方
- 各自选角色 → 窗口 A 点 Start Game → 双方进入 Battle
