# 搭建 01 — 创建 4 个空场景

1. 打开 Cocos Creator
2. 左下角 **资源管理器** → 右键 `assets` → **新建** → **Scene**
3. 重命名为 `Home`（不要输入 `.scene` 后缀，编辑器自动加）
4. 重复步骤 2-3，再创建三个场景，分别重命名为 `Lobby`、`Battle`、`Roguelite`

完成后 `assets/` 下应该有：

```
Home.scene
Lobby.scene
Battle.scene
Roguelite.scene
```

> 每一个场景双击打开后，Cocos 会自动在层级管理器里生成 Canvas + Camera 节点。不用删它们。
