# 搭建 04 — 配置 Build Settings

1. 顶部菜单 → **项目** → **构建发布**
2. 在 **参与构建场景** 区域，确保列表里有 4 个场景，且 **Home 在最上面**
3. 场景名必须严格等于（大小写敏感）：

| 场景文件 | 场景名 |
|---|---|
| Home.scene | Home |
| Lobby.scene | Lobby |
| Battle.scene | Battle |
| Roguelite.scene | Roguelite |

> 如果名字不对：资源管理器里单击选中场景文件 → 右侧属性检查器里修改场景名。

4. 关掉面板即可，不需要真的构建
