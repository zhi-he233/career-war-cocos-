# 搭建 02 — 给 4 个场景挂脚本

不需要手动建任何 UI 节点。场景脚本的 `onLoad` 里会自动生成所有 Label 和 Button。

### Home 场景

1. 双击 `Home.scene`
2. 选中 **层级管理器** 里的 `Canvas`
3. 右侧 **属性检查器** → 底部 **添加组件** → **自定义脚本** → **HomeScene**
4. 在 HomeScene 组件里确认 `Server Url` 是 `http://localhost:3001`
5. `Ctrl + S` 保存

### Lobby 场景

1. 双击 `Lobby.scene`
2. 选中 `Canvas` → **添加组件** → **自定义脚本** → **LobbyScene**
3. `Ctrl + S` 保存

### Battle 场景

1. 双击 `Battle.scene`
2. 选中 `Canvas` → **添加组件** → **自定义脚本** → **BattleScene**
3. `Ctrl + S` 保存

### Roguelite 场景

1. 双击 `Roguelite.scene`
2. 选中 `Canvas` → **添加组件** → **自定义脚本** → **RogueliteScene**
3. `Ctrl + S` 保存
