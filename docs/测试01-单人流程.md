# 测试 01 — 单人创建房间 + 战斗

> 前置：服务器 `localhost:3001` 在运行，搭建 01-04 已完成

### 1. 打开 Home 场景预览

双击 `Home.scene` → 点顶部 ▶️ 预览

### 2. 创建房间

点 **经典混战** 按钮。

状态栏应依次显示：
```
Connecting http://localhost:3001
Connected ...
Room <房间号> | lobby
```

然后自动跳到 Lobby 场景。

### 3. Lobby 选角色

- 点角色网格里的任意角色按钮（如 Boxer）
- 点技能按钮选召唤师技能
- 选中项前面会显示 `>`

### 4. 开始游戏

点 **Start Game** 按钮。应自动跳到 Battle 场景。

### 5. 战斗

- 看到双方玩家状态行
- 选中一个目标（点目标按钮）
- 点 **Roll** 按钮 → 显示骰点
- 点 **Attack** 按钮（或其他行动按钮）→ 行动结算
- 敌人自动回击

### 6. 状态栏信息

- `Connected ...` ：网络已连接
- `Room <id> | <phase>` ：当前房间状态
- `<event> ok` ：操作被服务器接受
- `<event> failed: ...` ：操作被拒绝，看错误原因
