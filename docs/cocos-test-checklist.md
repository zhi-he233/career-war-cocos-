# Cocos Test Checklist

Use this file as a simple smoke-test guide for the current graybox migration.

## Before Preview

1. Start the backend server.
   - Expected URL: `http://localhost:3001`
   - Socket.IO browser script should be available at:
     `http://localhost:3001/socket.io/socket.io.js`
2. Open `assets/scenes/Home.scene`.
3. Press Preview.

## Single Client Flow

1. Click `Classic`, `PVE 1v1`, or `Roguelite`.
2. Home should show status text:
   - `Connecting http://localhost:3001`
   - `Connected ...`
   - `Room <id> | lobby`
3. The app should route to `Lobby`.
4. In `Lobby`, choose a character and a summoner skill.
5. Click `Start Game`.
6. The app should route to `Battle` for classic/PVE combat.
7. In `Battle`, select a target if needed, then click `Roll`.
8. After the roll, choose an action button.

## Two Client Join Flow

1. Open Preview window A.
2. Click `Classic`.
3. In `Lobby`, copy/read the room id from the status line.
4. Open Preview window B.
5. Either type that room id into `Room ID`, or click `Refresh Rooms` and choose the room from the list.
6. Click `Join Room` if you typed the id manually.
7. Window B should route to the same `Lobby`.

## Roguelite Flow

1. Click `Roguelite` from Home.
2. Start the game from Lobby.
3. Battle works through the normal Battle scene.
4. After combat, the server can route to `Roguelite`.
5. The graybox Roguelite scene supports:
   - reward choices
   - event choices
   - shop buy and leave
   - rest actions
   - route continue

## What The Status Line Means

- `Connected`: Socket.IO is connected.
- `Room <id> | <phase>`: room state was received.
- `<event> ok`: server accepted a button action.
- `<event> failed: ...`: server rejected the action. Read this first before checking the console.
- `Network error: ...`: backend server or Socket.IO script is not reachable.

## Current Graybox Limits

- UI is generated by scripts, not final art prefabs.
- Buttons and labels are functional placeholders.
- Character and skill names are English fallback names to avoid encoding issues.
- Roguelite map is a simple route list, not the final visual map.
